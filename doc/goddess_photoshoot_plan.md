# AI 女神约拍神器 (V1 精简版) 开发方案

## 一、 项目背景与核心价值

### 1.1 背景
本项目旨在利用 **SupaFast (Vue + Supabase + FastAPI + R2)** 架构，结合中间商提供的 GPT-image2 接口，快速构建一个面向公众号粉丝的“H5 智能影楼”。

### 1.2 核心策略：极致精简 (Lean Startup)
- **低成本**：不自建 GPU 服务器，不开发复杂的 AI 算法，直接集成成熟 API。
- **低风险**：通过“固定模板”锁定输出内容，规避违规 Prompts 风险。
- **快速迭代**：先上线跑通现金流，盈利后再进行合规升级与技术深化。

---

## 二、 核心功能设计

### 2.1 镜像写真 (仅限固定模板)
- **输入**：用户上传 1 张正面人像。
- **逻辑**：后端根据用户选择的模板 ID，调用预设好的“极致颗粒度”Prompt。
- **模板示例**：
    - [模板A]：影楼风 - 极简白背景，法式复古婚纱。
    - [模板B]：职场风 - 办公室背景，都市精英西装。
    - [模板C]：度假风 - 海滩落日背景，波西米亚长裙。

### 2.2 极简商业化
- **支付方式**：购买积分卡密（对接面包多等平台）。
- **兑换逻辑**：H5 输入卡密 -> Supabase 更新积分 -> 消费积分开启任务。

---

## 三、 安全与合规保障 (V1 方案)

### 3.1 法律免责（第一道防线）
- **强制勾选声明**：上传前用户必须勾选：“我已确认照片为本人或已获授权，且仅用于个人合法娱乐”。
- **禁止性提示**：页面显著位置标注：“严禁上传明星、政治人物等敏感照片”。

### 3.2 技术标识（合规底线）
- **自动压制水印**：FastAPI 在获取 AI 返图后，利用 Python `Pillow` 库自动在图片角落压入“AI 合成 / AI Generated”字样。
- **隐私保护**：用户上传的原图在生成任务结束后（或 1 小时内）从 R2 彻底删除，不作保留。

### 3.3 借力过滤（依赖上游）
- **原生安全盾**：利用 GPT-image2 接口自带的内容过滤机制，拦截不合规的输入与输出。
- **任务熔断**：在数据库设置 `service_active` 状态位，遇紧急情况可一键关停服务。

## 五、 开发路线图 (3天)

- **Day 1: 业务骨架**：完成 Supabase 积分表设计与 Vue 基础上传页面。
- **Day 2: AI 集成**：对接中间商 API，实现“上传-生成-加水印-存入R2”全链路。
- **Day 3: 闭环与上线**：配置卡密充值，在公众号进行粉丝小规模测试。
